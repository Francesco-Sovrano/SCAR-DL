# SCAR-DL

## Installation

* requires python 3.7+ *
* clone this repo
* cd to the repo
* start jupyter notebook: `jupyter notebook`
* open scar.ipynb: `http://localhost:8888/notebooks/scar.ipynb`

**Tested on macOS Mojave and Catalina**
